# Why is it so simple?

Complicated does not mean awesome.

# How do I dev it?

Open four consoles:
* `npm start` starts the server
* `npm run browser-sync` wires up chrome
* `gulp watch` compiles and deploys changes
* `npm run testwatch` runs the tests in watch mode
